cl_khr_int64_base_atomics

Optional extensions that implement atomic operations on 64-bit signed and unsigned integers to locations in global and local memory.

#pragma OPENCL EXTENSION cl_khr_int64_base_atomics : enable

Description

The behavior of these extensions is defined in the SPIR-V environment and appropriate kernel language specifications.

Also see

EXTENSION, cl_khr_int64_extended_atomics, Atomic Functions

Specification

OpenCL 2.1 Extensions Specification, page 12
