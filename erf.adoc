[[erf]]
= erf, erfc

Error functions.

[source,c]
----
gentype erf(gentype)

gentype erfc(gentype)
----


[[description]]
== Description

`erf` is the error function encountered in integrating the normal distribution.

`erfc` is the complementary error function.
