get_group_id

Returns the work-group ID.

size_t get_group_id(uint dimindx)

Description

Returns the work-group ID which is a number from 0 .. get_num_groups(dimindx) - 1. Valid values of dimindx are 0 to get_work_dim() - 1. For other values, get_group_id() returns 0.

Also see

Work-Item Functions, clEnqueueNDRangeKernel
